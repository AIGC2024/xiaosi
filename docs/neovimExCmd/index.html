Ex 命令行

在普通模式使用 : 后neovim会把光标移动到窗口下方，在该模式下可执行 Ex 命令。

命令行编辑

快捷键 说明
<Insert> 在插入和替换之间切换
<CTRL-C 不执行命令，退出命令行模式
<CTRL-[>
或 <Esc>		 退出命令行模式
<Left> 光标左移
<S-Left>
或 <C-Left>		 光标左移一个字串 (WORD)
<CTRL-B>
或 <Home>		 光标移动至命令的起始
<Right> 光标右移
<S-Right>
或 <C-Right>		 光标右移一个字串 (WORD)
<CTRL-E>
或 <End>		 光标移动至命令的末尾
<CTRL-H>
或 <BS>		 删除光标前面的字符
<CTRL-W> 删除光标前的单词 ( word )。单词范围受 iskeyword 选项影响
<CTRL-U> 删除光标所在位置与行首之间的所有字符
<Del> 删除光标后面的字符
<CTRL-V> 下一个字符如果是非数字字符，按本义插入。
或不超过三个的数字字符可用于输入单个字节的十进制编码。这里的非数字和三个数字字符不经过映射
<CTRL-Q> 同 <CTRL-V> 但在终端环境没效果
<CTRL-SHIFT-V>
或 <CTRL-SHIFT-Q>		 同 <CTRL-V>
{char1} <BS> {char2}
或 <CTRL-K> {char1} {char2}		 输入二合字母，如果 {char1} 是一个特殊键，插入该键 <> 形式的编码
<CTRL-R><CTRL-F> 插入光标下的文件名
<CTRL-R><CTRL-P> 插入光标下的文件名与路径
<CTRL-R><CTRL-W> 插入光标下的单词
<CTRL-R><CTRL-A> 插入光标下的字串
<CTRL-R><CTRL-L> 插入光标下的文本行
<CTRL-R> {register} 插入寄存器里的内容。当按下 <CTRL-R> 时，屏幕会显示一个 " 字 符，提示你输入一个寄存器的名字。映射、缩写、补全功能不会生效，
但 <BS> <br> <CTRL-W> 仍旧可能结束命令行模式，并使剩余的字符被接下来其他的模式解释
` {
<CTRL-\> e {expr} 使用表达式
<Up> 在历史记录中查找开始部分与当前输入匹配的前一条命令
<Down> 在历史记录中查找开始部分与当前输入匹配的下一条命令
<CTRL-Y> 当存在无模式选择时，复制该选择区域的内容至剪贴板
<CTRL-]> 激活缩写，但不插入任何字符
<CTRL-M>
或 <CTRL-J>
或 <CR>
或 <NL>		 执行输入的命令
