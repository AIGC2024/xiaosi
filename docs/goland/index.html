使用

  • goroot 是指 golang sdk 目录
  • gopath 是指 golang 工作目录，新版本不用配置，会与 go mod 冲突

error

$GOPATH/go.mod exists but should not

错误详情

原因:开启Go module模块支持后，并不能与$GOPATH共存,所以把项目从$GOPATH中移出或者不要使用Go module模块即可

方法1：删除 gopath

方法2：删除 go.mod 文件
