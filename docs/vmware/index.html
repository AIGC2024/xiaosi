共享主机目录

虚拟机关机状态下执行以下操作

  1. 选择使用共享文件的虚拟机
  2. 编辑虚拟机设置
  3. 选项卡
  4. 共享文件夹
  5. 总是启用
  6. 添加
  7. 安照引导添加共享的文件夹
  8. 保存

在虚拟机器中安装 open-vm-tools

1
2
3
4
5

[root@master ~]# yum install -y open-vm-tools
Loaded plugins: fastestmirror
Loading mirror speeds from cached hostfile
 * epel: my.mirrors.thegigabit.com
...

检查是否识别到共享的本地文件夹

1
2

[root@master ~]# vmware-hgfsclient 
note

创建挂载目录，如果该目录可能已经存在，不用重新创建

1

[root@master ~]# mkdir /mnt/hgfs/

挂载共享文件夹

1

[root@master ~]# vmhgfs-fuse .host:/ /mnt/hgfs -o uid=0 -o gid=0 -o umask=022 -o allow_other
  • .host:/ ：表示本地共享文件的更目录
  • /note ： 表示虚拟机的挂载点

查看挂载

1
2

[root@master ~]# ls /mnt/hgfs/
note

开机自动挂载

1
2

[root@master ~]# chmod +x /etc/rc.d/rc.local
[root@localhost ~]# echo 'vmhgfs-fuse .host:/ /mnt/hgfs -o uid=0 -o gid=0 -o umask=022 -o allow_other' >> /etc/rc.d/rc.local

docker无法挂载vmware共享目录

挂载时加-o allow_other（允许其它用户访问）

克隆注意事项

  • 修改主机名
  • 修改网卡的 UUID 与 IP 地址（如果冲突）
