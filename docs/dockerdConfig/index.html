防火墙规则

dockerd 命令行选项：

1
2

--ip6tables   # 自动添加 ip6tables 规则
--iptables    # 自动添加 iptables 规则，默认值为 true

/etc/docker/daemon.json 配置项：

1
2
3
4
5

{
  ...
  "iptables": false,
  ...
}

禁止 docker 添加防火墙规则会导致有以下问题：

  • 使用 bridge 类型的容器将无法访问主机服务，需要手动添加防火墙规则
