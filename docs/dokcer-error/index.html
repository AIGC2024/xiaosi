docker 容器使用中遇到的错误

小厮 收录于 容器
2023-02-21 2024-01-09 约 800 字 预计阅读 2 分钟 
目录
  • error initializing graphdriver
  • error initializing graphdriver: overlay2: the backing xfs filesystem is formatted without d_type support
  • unable to configure the Docker daemon with file /etc/docker/daemon.json: invalid character ‘"’ after object key:value pair

运行环境：

  • centos: 7

内容来自以下文档：

  • Deprecated Engine Features
  • xfs-filesystem-changing-ftype-0-to-ftype-1

error initializing graphdriver

报错信息

1

failed to start daemon: error initializing graphdriver: prior storage driver devicemapper is deprecated and will be removed in a future release; update the the daemon configuration and explicitly choose this storage driver to continue using it; visit https://docs.docker.com/go/storage-driver/ for more information

解决方法，修改使用的驱动程序

1
2
3
4
5
6

[root@localhost ~]# cat /etc/docker/daemon.json | jq .
{
  "data-root": "/docker/",
  # 修改为 overlay2
  "storage-driver": "overlay2"
}

原因：docker 使用 overlay2 替代了 devicemapper

error initializing graphdriver: overlay2: the backing xfs filesystem is formatted without d_type support

{
  # 指定目录  
  "data-root": "/docker/",
  "storage-driver": "overlay2"
