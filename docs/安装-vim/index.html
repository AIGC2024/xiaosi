vim 与 neovim

Vim是一个功能强大的文本编辑器，已有20多年历史， Vim 已经积累了大约 30 万行可怕的 C89 代码。 导致程序的维护、Bug 的 修复、以及新特性的添加都变得越来越困难。

巴西程序员 Thiago de Arruda Padilha（aka tarruda）向 Vim开源编辑器项目递交了两大补丁，对Vim的架构进行了大幅调整，结果遭到了Vim作者Bram Moolenaar的拒绝，因为对于Vim这样一个成熟的项目进行如此大的改变风险太高。于是tarruda发起了Vim fork项目Neovim

简单说，neovim是vim重构版本，两者目前活跃的人也很多
