运行环境： centos: 7

安装相关的软件包

避免出现编译出错，无法编译，升级之后无法使用等情况

1

[root@master ~]# yum install -y zlib* readline* gcc*  libffi-devel

从Python官网下载python3.7.3的安装包

1

[root@master ~]# wget https://www.python.org/ftp/python/3.7.3/Python-3.7.3.tgz

[root@master ~]# tar xf Python-3.7.3.tgz 

[root@master ~]# cd Python-3.7.3

[root@master Python-3.7.3]# mkdir /usr/local/python3.7.3

[root@master Python-3.7.3]# ./configure --prefix=/usr/local/python3.7.3

[root@master Python-3.7.3]# make

[root@master Python-3.7.3]# make install

创建软连接

1
2
3
4
5
6

[root@master Python-3.7.3]# mv /usr/bin/python /usr/bin/python_2.7.5

[root@master Python-3.7.3]# ln -s /usr/local/python3.7.3/bin/python3.7 /usr/bin/python

[root@master Python-3.7.3]# python -V
Python 3.7.3

还原到原版本

1
2
3
4
5
6

[root@master ~]# mv /usr/bin/python /usr/bin/python_3.7

[root@master ~]# mv /usr/bin/python_2.7.5 /usr/bin/python

[root@master ~]# python -V
Python 2.7.5

解决升级之后python之后yum无法使用问题

将 /usr/bin/yum 和 /usr/libexec/urlgrabber-ext-down 这2个文件的第一行指向 /usr/bin/python_2.7.5
